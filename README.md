# ドメイン駆動設計入門

## 要求

- ユーザ管理
- サークル管理

## 仕様

- ユーザ管理
  - ユーザID,ユーザ名,メールアドレス
  - ユーザIDは3文字以上32文字以下
  - ユーザ名は3文字以上20文字以下
  - メールアドレスの重複は禁止
  - ユーザの登録,更新,削除

- サークル管理
  - サークルの作成
  - サークルへの参加
  - サークル名は3文字以上20文字以下
  - サークル名の重複禁止
  - サークルには参加者の中から(一名以上?)サークルオーナーを設定
  - サークルへの参加人数はオーナー含めて30人以下
